Input group:


@

Acima foi adicionado um input type text com class 'form-control' e placeholder 'e-mail',em cima dele coloquei uma div com class 'input-group-text' onde introduzi o texto do '@',e envolvendo esses dois componentes coloquei uma div com class 'input-group',que deu o resultado de unir os dois componentes do nosso form.


@

Acima foi a seguinte combinação: div class 'input-group' + (dentro da div) div class 'form-control' e placeholder 'Usuário' + div class 'input-group-text' '@' + div class 'form-control' e placeholder 'Domínio'.


@

Acima eu mesclei dentro da div class 'input-group': uma (div class "input-group-text") + (input type'text' class'form-control') + (button class 'btn-primary').


Lembrando que se você desejar alterar o tamanho basta adicionar um "input-group-sm ou (lg)" na div que está abraçando e envolvendo todos os componentes como está acima.
